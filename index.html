Your Responsibilities

Each week, you will be expected to:

  • Attend every class and complete any practice activities assigned during the lecture. Textbook reading before class has proven to significantly enhance student learning. After the class, complete the assigned textbook reading while looking over the related source code closely. You should also review slides & source code from the reference book along with ALL of the posted resources. Then start working on the assignment for the week. Attend office hours to clarify concepts. Use piazza to ask questions about everything else.
  • It is also your responsibility to be familiar with all course policies including the ones about Academic Integrity.
